# Redesain UI/UX Panel Hatchery PT Prima Larvae

## Goal
Mendesain ulang antarmuka Panel Hatchery PT Prima Larvae agar nyaman untuk pemantauan data kualitas air & larva, efisien dalam pemakaian operasional harian, serta tampil modern dan intuitif.

## Tasks
- [ ] Task 1: Update sistem desain warna, tipografi, dan komponen dasar (`css/styles.css`, `index.html`) → Verify: Buka aplikasi di browser, pastikan palet warna brand dan kartu elevated tampil konsisten.
- [ ] Task 2: Lengkapi pustaka ikon SVG (`js/icons.js`) → Verify: Ikon filter, search, stepper, dan tren tampil tanpa error render.
- [ ] Task 3: Tambahkan Quick Search & Top Header Bar (`js/app.js`) → Verify: Header baru menampilkan info peran, pencarian cepat, dan tombol reset/logout dengan responsif.
- [ ] Task 4: Perbarui Dashboard Beranda dengan KPI Widgets & Peringatan Cepat (`js/views/dashboard.js`) → Verify: Kartu KPI dapat diklik untuk memfilter data, peringatan bahaya/waspada tampil dengan rekomendasi aksi.
- [ ] Task 5: Tingkatkan visualisasi Grafik Tren (`js/components/chart.js`) → Verify: Grafik air dan larva menggunakan area gradient fill, tooltip status batas, dan filter periode waktu.
- [ ] Task 6: Tambahkan Visual Stepper Alur Verifikasi & Toolbar Filter (`js/components/recordList.js`, `js/components/table.js`, `js/views/hub.js`) → Verify: Status verifikasi *Draft → Ka.Sie → QC → MPM* tampil sebagai stepper visual, tabel data mudah difilter per-tank.

## Done When
- [ ] Antarmuka panel lebih tajam, nyaman dipandang, dan data mudah dipahami dalam sekali lihat.
- [ ] Alur kerja pencatatan dan verifikasi data menjadi lebih cepat dan efisien.
- [ ] Seluruh fungsi navigasi, filter, dan login 9 peran demo berjalan lancar tanpa kendala.
